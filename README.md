# Rakuten Grid

## Simple Syntax

```bash
<div class="rk-container">
    <div class="rk-row">
        <div class="col-m-12 col-d-6">
            Your code here
        </div>

        <div class="col-m-12 col-d-6">
            Your code here
        </div>
    </div>
</div>

```
